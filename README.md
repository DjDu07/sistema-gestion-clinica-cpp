# Sistema de Gestión Clínica - Clínica Psicológica "Volver a vivir"

Sistema orientado al almacenamiento de datos y la creación de reportes estadísticos y demográficos desarrollado en **C++** utilizando Visual Studio.

## 🚀 Características del Sistema
- **Módulo de Pacientes:** Registro validado por DNI único de 8 dígitos.
- **Módulo de Administración:** Gestión de códigos de terapias, precios, duraciones e internamientos.
- **Reportes Analíticos:**
  - Gráfico de barras interactivo de recaudación financiera por tipo de adicción (videojuegos, drogas, alcohol).
  - Cálculo demográfico de porcentajes de internamiento segmentado por rangos de edad (10-17, 18-30, 31-60).
  - Listado dinámico descendente de pacientes ordenados por apellido.
  - Reporte analítico de promedio de edad de pacientes registrados por cada sede local.

## 🛠️ Tecnologías Utilizadas
- **Lenguaje:** C++ Estándar
- **IDE:** Visual Studio
- **Conceptos clave:** Punteros, Estructuras de Datos (`struct`), Asignación dinámica de memoria y Estructuras de control condicionales/cíclicas.
